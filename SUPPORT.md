# Support

## How to file issues and get help  

This private preview uses GitHub Issues to track bugs and feature requests.

Please search existing issues before filing a new one.

When creating an issue, include:

- VS Code version
- Extension version under test
- OS and architecture
- Reproduction steps
- Prompt and generated query (sanitized)
- Expected behavior and actual behavior
- Error logs/screenshots if available

For preview onboarding questions or customer-specific coordination, contact the Cosmos DB preview program owner through your existing Microsoft engagement channel.

## Microsoft Support Policy  

Support for this project is limited to the resources listed above.
